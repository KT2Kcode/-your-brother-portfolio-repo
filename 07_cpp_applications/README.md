### 07_cpp_applications

* **Goal**: Developed high-performance console applications using C++ to master core algorithms, computer memory management, and data structures.

* **Projects Included**:
  * **01_player_roster**: Engineered an object-oriented tracking system to manage, update, and display athlete profiles using custom classes and vectors.
  * **02_guessing_game**: Formulated a logic-based terminal game that implements random number generation, input validation, and conditional tracking loops.
  * **03_array_board**: Constructed a multi-dimensional array-based data grid simulating a game board or matrix to optimize localized memory access.

* **Key Skills Used**: Object-Oriented Programming (OOP), Data Structures (Arrays/Vectors), Memory Management, and Algorithmic Logic.
