### 06_web_applications

* **Goal**: Developed responsive front-end web components and interactive layouts using HTML, CSS, and dynamic JavaScript injection.

* **Projects Included**:
  * **01_index_test**: Formulated semantic HTML structures and DOM architecture tests to optimize cross-browser compatibility.
  * **02_news_gallery**: Built a dynamic, media-focused grid layout utilizing JavaScript to handle interactive news and image rendering.
  * **03_copyright_component**: Engineered a reusable script-driven footer component that utilizes JavaScript to dynamically pull and update server-side calendar years for copyright compliance.

* **Key Skills Used**: Document Object Model (DOM) Manipulation, Semantic HTML5, CSS Grid/Flexbox Layouts, and Event Handlers.
